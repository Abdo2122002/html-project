Summary Of (PL)

Programming:
Something somewhat intangible compared to other products,
a series of thousands or millions of commands that ask a computer to perform certain operations such as displaying information,
performing calculations, or storing data.
This software is like the soul of the body in the computer system
and it is constantly expanding and increasing in complexity, requirements and tasks that it performs.



As for software engineering:
it is a branch of engineering based on a set of foundations and rules aimed at designing and developing programs in abundance and high quality that meet the needs of users.
It is not enough to find good integrated software the work of one person, but it requires a team of good engineers.
It was necessary to find a science concerned with software engineering to lay the foundations and standards that protect this profession from intruders so
that it is possible to distinguish good software from bad ones, and software verification in software project management, software testing and software engineering,
as well as to indicate the audit mechanism that the software project conforms to the specified specifications and that it It works the way expected.
The term may also be used to control program quality. Usually,
this task is one of the tasks of the software engineer who performs the software testing process,
